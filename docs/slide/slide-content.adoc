== はじめに

本スライドはAsciiDocで作成したサンプルです。 +
主にPlantUMLの図の例を載せています。

PlantUMLとは::
PlantUMLはUMLなどの図をテキストで記述できるツールです。 +
下記の公式サイト(日本語)にサンプルが豊富に掲載されているので参考になります。 +
https://plantuml.com/ja/ +
このスライドでも上記サイトのサンプルを一部変更したものを使用しています。
